Implementation Notes

  • learned angular from codeschool.com's free course, "Shaping up with Angular JS"
  • I think I started it as a JetBrains Webstorm empty project
    • used npm to make a package.json
    • I used npm to install bootstrap, angular, n3-charts, chai, express, mocha. Used the --save to write it to the package.json
  • created a local git repository
  • created a github repository and copied my code to it (doesn't copy node_modules)
  • I used n3-charts for the graphing package. Could probably substitute something else fairly easily. Maybe would need to change the data structure.
  • the n3-chart was resizing when anything was clicked. This was solved by creating a style so I could set the size of class "my-chart"
